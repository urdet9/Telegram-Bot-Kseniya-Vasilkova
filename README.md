# Telegram Bot for Kseniya Vasilkova

Выпускной проект по курсу **«Промпт-инжиниринг 2.0»**.

## 📌 О проекте

Telegram-бот автоматизирует работу фитнес-нутрициолога:

- AI-мини-тест;
- продажу цифрового гайда;
- приём заявок на консультацию;
- приём заявок на сопровождение;
- хранение данных в Google Sheets;
- уведомления владельца проекта.
